Countries and Territories

Provides a list of countries and territories for use in DIT services and reporting. The list is a combination of two published gov.uk registers managed by Foreign and Commonwealth Office - the country register and the territory register. These registers can be accessed below:

- British English names of all countries currently recognised by the UK government
Country register (www.registers.service.gov.uk/registers/country)

- British English names of territories not currently recognised as countries by the UK government
Territory register (www.registers.service.gov.uk/registers/territory)

Field definitions

Name
The commonly-used name of a record
Official name
The official or technical name of a record
Type
The source of a record - gov.uk Country or gov.uk Territory
Key
The 2 letter ISO 3166-2 alpha2 code (for countries) or the ISO 3166-1 alpha3 code (for territories). Unique codes have been created for territories that don't have an existing ISO code
Start date
The date a record first became relevant to a register
End date
The date a record stopped being applicable
