What is Data Workspace?

The Department's Data Catalogue shows collections of the different data sets that the department generates, uses, publishes and holds - the purpose of this is to increase availability and visibility of data within the department. A dataset can take many forms (tables, spreadsheets, database reports) but is essentially a structured set of data.

Some of the data will be available to download, other data sets will require that you request access.

The data will also be accessible to use with certain tools for visualisation and analysis. There is more information about the tools here.
